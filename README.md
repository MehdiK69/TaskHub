# TaskHub

Application de gestion de tâches développée en Java dans le cadre d'un projet portfolio.

## Stack technique
- **Langage** : Java
- **Interface graphique** : Swing
- **Architecture** : MVC
- **Tests** : JUnit
- **Persistance** : Fichiers

## Fonctionnalités
- [x] Création / modification / suppression de tâches
- [x] Catégories de tâches (énumérations)
- [x] Persistance des données en fichier
- [x] Tests unitaires JUnit (à finir)
- [ ] Interface graphique Swing (en cours...)

## Architecture
Le projet suivra le pattern MVC :
- **Modèle** : gestion des tâches et de la persistance
- **Vue** : interface Swing
- **Contrôleur** : logique applicative
